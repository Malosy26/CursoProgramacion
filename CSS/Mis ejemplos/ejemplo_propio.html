Comandos Git

Tabla de Comandos Básicos de Git
Comando Función Comentarios
git init Iniciar repositorio Úsalo una sola vez al comenzar un proyecto
git clone "repo" Clonar un repositorio existente Necesitas la URL del repositorio remoto
git status Ver estado de los archivos Úsalo frecuentemente para saber qué cambió
git add . Añadir todos los archivos al staging Incluye todos los cambios sin seleccionar
git add "elemento" Añadir un archivo específico al staging Ideal para hacer commits más limpios
git commit -m "string" Crear un commit con mensaje El mensaje debe describir claramente el cambio
git commit -a -m "mensaje" Hacer commit directo de archivos modificados No incluye archivos nuevos no añadidos antes
git push Subir cambios al repositorio remoto Necesario para compartir tu trabajo
git pull Traer y fusionar cambios del remoto Úsalo antes de empezar a trabajar cada día
git remote add Agregar repositorio remoto Ejemplo: git remote add origin URL
git remote -v Ver repositorios remotos configurados Muestra las URLs asociadas al proyecto
git fetch Descargar cambios remotos sin fusionar Seguro si solo quieres revisar antes de fusionar
git merge Fusionar ramas Requiere resolver conflictos si existen
Comandos Ramas
Comando Función Comentario
git branch "rama" Crear una nueva rama Permite trabajar en una línea de desarrollo separada
git checkout "rama" Cambiar a otra rama Úsalo para moverte entre ramas existentes
git merge rama Fusionar ramas Combina los cambios de una rama con la actual
